# Basic Node.js with ECMAScript 2015 (ES6)

This's a managing tools for travel, helpful travellers and make a wonderful trip.

Libraries integrated:
- Node.js 4.x.x
- Express.js 4.x
- ECMAScript 2015 (ES6)

### Installation

```sh
$ git clone [git-repo-url] nodejs
$ cd nodejs
$ npm node-dev -g
$ npm install
$ node-dev server.js
```
